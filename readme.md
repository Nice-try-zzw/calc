## 这是一个炒鸡简单的计算器程序
基于C++  
主干代码不超过100行  
使用的结构是栈  
利用QT简单的创建了一个界面  没有太多的功能  
目前只是实现了整数上的操作  
可以支持加减乘除  
&&、||、()操作  
现在支持将合法算式复制到文本框中 进行运算
